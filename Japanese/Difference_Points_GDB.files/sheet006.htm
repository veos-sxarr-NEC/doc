「アドバンスオフ」モードの設定セッテイの仕方シカタ
このシートでは、GDBを使用シヨウして、VEプログラムをアドバンスオフモードで実行するための説明がされています。
このモードでは前回ゼンカイの命令メイレイが完了カンリョウするまで命令メイレイ実行ジッコウが保留ホリュウされます。
その結果ケッカ、GDBはHW例外レイガイが生ショウじるときの正確セイカクな状態ジョウタイを示シメします。
「アドバンスオフ」モードはGDBでは下記カキの手順テジュンにより設定セッテイされます。
1. ブレイクポイントをメイン()ファンクションに設定セッテイします。
2. VEプログラムを実行ジッコウします。VEプログラムはメイン()ファンクションで停止テイシします。
3. PSWレジスタのアドバンスオフモードビットを設定セッテイします。
set $psw=($psw | 0x8000000000000000)
4. VEプログラムを継続ケイゾクします。

(注意チュウイ) 　"segfaut_g"はセグメンテーションフォルトを引ヒき起オこすVEプログラムです
