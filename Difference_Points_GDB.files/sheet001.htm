Difference Points for GDB
Revision 2.5
NEC
This document describes differences in usage of GDB between that for VE(Vecor Engine) and general Linux.
This document covers VEOS v2.3 or later.
This document consists of the following sheets:
Summary
Summary of difference points of GDB for VE.
VE specific commands
List of new GDB commands prepared for VE.
GDB commands
List of GDB commands with the current status of support for each command.
MI commands
List of GDB/MI commands with the current status of support for each command.
How to set advance off mode
Descriptions to run a VE program in "advance off" mode uner GDB.
Revision History
For detail of usage for each command, please refer to "GDB User Manuual" in GNU project.
GDB is developed by GNU project.
GDB for VE is developed based on GDB 7.12.1 by NEC.
